## ⚖️ Sobre o Projeto

Advogados e profissionais jurídicos perdem tempo tentando formular o prompt certo para obter uma resposta útil de uma IA. O **Cássia Prompt** resolve isso: basta preencher três campos — o que gerar, o objetivo e os fatos do caso — e o sistema entrega um comando estruturado, com diretrizes claras para que a IA use lei seca e jurisprudência real, sem inventar nada.

Criado para uso prático no dia a dia jurídico, com visual premium e foco total em produtividade.

---

## ✨ O que o sistema faz

- **Gerador de prompts jurídicos** — monta automaticamente um comando estruturado com base nos dados preenchidos
- **Importação de PDF** — extrai o texto de documentos jurídicos diretamente no navegador, sem upload para servidor
- **Barra de integridade dos dados** — indica visualmente o quanto o prompt está completo antes de gerar
- **Exportação em Word (.doc)** — baixa o prompt gerado como documento editável
- **Cópia rápida** — copia o prompt com um clique para colar em qualquer IA
- **Modo claro/escuro** — interface adaptável à preferência do usuário
- Roda 100% no navegador, sem cadastro, sem servidor, sem internet obrigatória

---

## 🛠️ Tecnologias

- **HTML, CSS e JavaScript puro** — zero dependências externas, carrega instantaneamente
- **PDF.js** (Mozilla) — leitura e extração de texto de PDFs direto no navegador
- **Blob API** — exportação do prompt como arquivo Word sem backend
